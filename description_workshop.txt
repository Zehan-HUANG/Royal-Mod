【宝饰王冠 (Jeweled Crown)】

"七颗宝石，七重祝福"

一顶镶嵌着七种珍贵宝石的华丽皇冠，
每颗宝石都蕴含着独特的力量。

▸ 属性加成：
  • +1 防御
  • +1% 伤害加成（全类型）
  • +1% 暴击几率
  • +1% 移动速度
  • +1% 攻击速度
  • +1 生命再生
  • -1% 魔力消耗

▸ 合成配方：
  在铁砧/铅砧处合成

  配方一：
  金皇冠 x1 + 琥珀 x1 + 黄玉 x1 + 钻石 x1 
  + 绿宝石 x1 + 蓝宝石 x1 + 紫晶 x1

  配方二：
  铂金皇冠 x1 + 琥珀 x1 + 黄玉 x1 + 钻石 x1 
  + 绿宝石 x1 + 蓝宝石 x1 + 紫晶 x1
